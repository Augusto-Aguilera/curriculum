# 🛠️ Patric Aguilera - Industrial Operative Profile

Este es el repositorio oficial del perfil profesional de **Patric Aguilera**, diseñado para destacar una trayectoria sólida en el sector industrial con una capacidad de innovación digital bajo el sello de **PATRIC SOFT ®**.

## 🚀 Propósito
El sitio funciona como una **verificación operativa industrial**, permitiendo a selectoras y empresas de mantenimiento visualizar experiencia técnica, formación avanzada y documentación legal en tiempo real.

## 🏗️ Stack Tecnológico
Para mantener una performance óptima y una estética industrial premium:
* **HTML5:** Estructura semántica avanzada.
* **CSS3:** Diseño responsivo con arquitectura de variables modernas y animaciones `cubic-bezier`.
* **Vanilla JavaScript:** Lógica de visualización dinámica para la "Bóveda Documental" (Soporte multi-imagen).

## 💎 Características Principales
* **Bóveda Documental:** Acceso directo a certificados, CUIL y antecedentes penales con visor integrado.
* **Galería en Planta:** Evidencia visual de operaciones reales con filtros de escala de grises interactivos.
* **Perfil Híbrido:** Sección dedicada a soluciones de software (**PATRIC SOFT ®**), demostrando versatilidad técnica.
* **Optimizado para Mobile:** Experiencia de usuario fluida en cualquier dispositivo.
